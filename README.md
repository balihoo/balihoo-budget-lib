A simple utility library to manipulate Balihoo marketing budgets.

## Usage
[See API docs](docs/API.md)

## Development

#### Requirements
```
sudo npm install -g gulp
npm install
```

#### Launch dev mode!
```
gulp
```

#### Generate docs
```
gulp docs
```
